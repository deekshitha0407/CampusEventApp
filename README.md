# CampusEventApp

CampusEventApp is an Android application designed to help students stay updated about campus events.

## Features
- View upcoming campus events
- Displays event name and date
- Simple and user-friendly interface

## Technologies Used
- Java
- Android Studio
- RecyclerView
- XML

## Project Structure
CampusEventApp
│
├── MainActivity.java
├── Event.java
├── EventAdapter.java
├── activity_main.xml
└── event_item.xml
